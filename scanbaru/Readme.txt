This PHP sample demonstrates how to upload acquired images to the file system on Web Server.

To run this sample, please make sure the write permission is granted to the 'SaveToFile' folder. Also, please place the MSI file in the same directory with these Web pages. The MSI file can be found in the installation directory of Dynamic Web TWAIN Plug-in. 

For more information on how to use Dynamic Web TWAIN Plug-in, please refer to:
http://www.dynamsoft.com/help/TWAIN/WebTwain/How%20to%20use%20Dynamic%20Web%20TWAIN%20Plug-in.htm

Should you need any technical help, please write to TWAINsupport@dynamsoft.com.

Thanks.
